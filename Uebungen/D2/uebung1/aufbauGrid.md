# Erklaerung unseres Grid-Layouts

Unser Grid ist mit CSS Grid Layout erstellt. Es besteht aus einem Container-DIV mit der Klasse `game-grid`, in dem sich mehrere `div`-Elemente mit der Klasse `game-item` befinden. Diese Elemente repraesentieren jeweils ein Videospiel.

## Struktur des Grid-Layouts

Im Container `game-grid` haben wir ein Grid-Layout definiert, das drei gleich breite Spalten enthaelt. Zwischen den Elementen des Grids haben wir einen Abstand von 20 Pixeln eingefuegt, um eine saubere und strukturierte Darstellung zu gewährleisten.

## Gestaltung der Spielelemente

- **Hintergrund und Abstaende:** Jedes Spielelement `game-item` hat einen hellgrauen Hintergrund, einen inneren Abstand von 20 Pixeln und zentrierten Text.
- **Bilder:** Die Bilder der Spiele haben eine feste Groesse, um ein einheitliches Erscheinungsbild sicherzustellen.
- **Schaltflaechen:** Die Schaltflächen „Kaufen“ innerhalb der Spielelemente sind grün, haben weisse Schrift und einen inneren Abstand, wodurch sie gut sichtbar und leicht anklickbar sind.

## Vorteile des Grid-Layouts

Unser CSS Grid Layout ermöglicht eine flexible und optisch ansprechende Anordnung der Spielelemente in einem strukturierten Raster, das sich automatisch an die Grösse des Containers anpasst.
